# Private Agentic AI Copilot for Warehouse Operations

This project is a private AI-powered warehouse copilot for an electronics e-commerce fulfillment warehouse. It uses a local LLM through Ollama and supports inventory queries, shipment queries, and warehouse document retrieval.

## Features
- Local AI using Ollama
- Inventory management with SQLite
- Shipment tracking
- Warehouse document upload and retrieval
- Simple agentic planner for tool selection
- Streamlit user interface

## Tech Stack
- Python
- Streamlit
- SQLite
- Ollama
- LangChain-Ollama
